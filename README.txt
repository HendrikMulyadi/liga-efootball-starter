Liga e-Football - PHP Native Starter Kit
---------------------------------------
Struktur file:
- index.php
- config.php
- functions.php
- teams.php
- edit_team.php (placeholder)
- delete_team.php (placeholder)
- generate_schedule.php
- matches.php
- update_result.php
- standings.php
- database.sql
- assets/css/style.css

Cara pakai:
1. Import database.sql ke MySQL (phpMyAdmin atau CLI).
2. Sesuaikan kredensial DB di config.php.
3. Letakkan folder ini di folder webserver (mis. htdocs atau www).
4. Buka teams.php untuk mulai menambah tim, lalu generate jadwal.
